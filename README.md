# Fuzzing AI Systems

Companion repository for the survey:

> **Fuzzing AI Systems: Foundations, Techniques, and Open Challenges**

This survey synthesizes **125 primary studies** identified within the
**January 2015–February 2026 search window** and examines:

- research trends and venue distribution;
- AI-system testing targets;
- fuzzing technique families;
- input-generation and mutation strategies;
- test-oracle types and construction mechanisms;
- reported failure types;
- recurring limitations, open challenges, and future directions.

## Companion Website

The companion website is available at:

https://mahmudakhatun.github.io/fuzzing-ai-systems-survey/

The website provides extended background, detailed methodology, study-level
mappings, taxonomy definitions, additional analyses, and discussion materials
that complement the main paper.
